# Superpowers 工作流（可选） / Superpowers Workflow (Optional)

当用户提出新功能、大改或「帮忙规划/设计」时：
When users request new features, major changes, or "help with planning/design":

1. 先检查 .cursor/skills/superpowers/ 下是否有适用技能（如 brainstorming、writing-plans、subagent-driven-development）。
   First check .cursor/skills/superpowers/ for applicable skills (e.g., brainstorming, writing-plans, subagent-driven-development).
2. 若适用，先读取对应 SKILL.md 并遵循其流程（例如先头脑风暴与设计确认，再拆任务与执行）。
   If applicable, read the corresponding SKILL.md and follow its process (e.g., brainstorm and confirm design first, then break down tasks and execute).
3. 与 ECC 的规则与技能并存：可同时使用 ECC 的 plan、tdd、code-review 等能力。
   Coexist with ECC rules and skills: you can use ECC's plan, tdd, code-review, etc. alongside.
