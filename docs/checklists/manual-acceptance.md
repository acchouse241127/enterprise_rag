# 手工验收清单

1. **登录**：打开前端，输入正确账号密码，应跳转首页；错误密码应提示失败。
2. **知识库**：进入知识库列表，应看到已有知识库（或空列表）；新建知识库（若已实现）应成功。
3. **上传**：选择知识库上传一个文档，应返回成功；文档状态应从 pending 变为 vectorized（或 parsing），可轮询或刷新列表查看。
4. **问答**：选择知识库、输入问题，应得到流式回答或降级检索结果；引用来源应可查看。
5. **看板**：进入检索质量看板页，应看到聚合数据或空状态，无报错。

## 通过标准

- 上述步骤无阻塞性错误；核心路径（登录 → 知识库 → 上传 → 问答）可走通。
