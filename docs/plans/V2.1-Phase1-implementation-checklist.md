## 接口变更摘要

### Before (旧接口)
```python
class BaseDocumentParser(ABC):
    def parse(self, file_path: Path) -> str: ...
```

### After (新接口)
```python
class BaseDocumentParser(ABC):
    def parse(self, file_path: Path) -> list[ParsedContent]: ...
    def parse_text(self, file_path: Path) -> str: ...  # 向后兼容
```
