## 技术要点

- BM25：使用 `ts_rank_cd` + `to_tsquery('jieba', ...)` + GIN 索引
- RRF：使用 `1 / (60 + rank)` 公式，K=60
- 中文递归分块：分隔符优先级 `\n\n` > `\n` > `。` > `；` > `，`
- Token 分块：tiktoken `cl100k_base` 编码
- 标题提取：Markdown `#` + 中文章节 `一、`、`1、`、`（1）`
- 父文档检索：
  - physical：通过 `parent_chunk_id` 查询 Parent chunk
  - dynamic：扩展相邻 chunks
  - off：不扩展
