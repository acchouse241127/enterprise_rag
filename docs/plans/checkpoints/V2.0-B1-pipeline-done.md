## 检索流程

```
用户查询
  ↓
[并行] BM25 检索 | 向量检索 (top_k * 2)
  ↓
RRF 融合（双路结果合并）
  ↓
父文档扩展（physical/dynamic/off）
  ↓
Reranker 重排序
  ↓
自适应深度截断（分数断崖检测）
  ↓
去噪过滤（Reranker + 关键词）
  ↓
最终结果 (top_k)
```

## 数据格式转换

### 第一次转换：为 RRF 融合准备

**BM25Result → dict（添加 bm25_score）**
```python
{
    "id": r.id,
    "document_id": r.document_id,
    "knowledge_base_id": r.knowledge_base_id,
    "chunk_index": r.chunk_index,
    "content": r.content,
    "section_title": r.section_title,
    "metadata": r.metadata,
    "bm25_score": r.bm25_score,
    "score": r.bm25_score,
}
```

**Vector Result → dict**
```python
{
    "id": r.get("chunk_id") or r.get("id"),
    "document_id": r.get("document_id", 0),
    "knowledge_base_id": r.get("knowledge_base_id", 0),
    "chunk_index": r.get("chunk_index", 0),
    "content": r.get("content", ""),
    "section_title": r.get("section_title"),
    "metadata": r.get("metadata", {}),
    "distance": r.get("distance", 0.0),
    "score": r.get("distance", 0.0),
}
```

### 第二次转换：RRF 融合结果

**RRFResult → RetrievalResult**
```python
RetrievalResult(
    id=r.id,
    document_id=r.document_id,
    knowledge_base_id=r.knowledge_base_id,
    chunk_index=r.chunk_index,
    content=r.content,
    section_title=r.section_title,
    metadata=r.metadata,
    score=r.rrf_score,
)
```

### 第三次转换：为 Reranker 准备输入

**RetrievalResult → dict**
```python
{
    "id": r.id,
    "content": r.content,
}
```

### 第四次转换：Reranker 输出

**Reranker Result → RetrievalResult**
```python
for rerank_result in reranked_results:
    for r in expanded:
        if r.id == rerank_result.get("id", ""):
            RetrievalResult(
                id=r.id,
                document_id=r.document_id,
                knowledge_base_id=r.knowledge_base_id,
                chunk_index=r.chunk_index,
                content=r.content,
                section_title=r.section_title,
                metadata=r.metadata,
                score=rerank_result.get("rerank_score", r.score),
            )
```

## 使用示例

```python
from app.rag.hybrid_pipeline import HybridRetrievalPipeline
from app.rag.bm25_retriever import BM25Retriever
from app.rag.retriever import VectorRetriever
from app.rag.rrf_fusion import RRFFusion
from app.rag.parent_retriever import ParentRetriever
from app.rag.adaptive_topk import AdaptiveTopK
from app.rag.denoiser import Denoiser
from app.rag.reranker import BgeRerankerService

# 初始化组件
bm25 = BM25Retriever()
vector = VectorRetriever(...)
rrf = RRFFusion(k=60)
parent = ParentRetriever()
adaptive = AdaptiveTopK(min_k=2, max_k=15, cliff_factor=1.5)
denoiser = Denoiser(reranker_threshold=0.15, keyword_overlap_min=0.2)
reranker = BgeRerankerService("BAAI/bge-reranker-v2-m3")

pipeline = HybridRetrievalPipeline(
    bm25_retriever=bm25,
    vector_retriever=vector,
    rrf_fusion=rrf,
    parent_retriever=parent,
    adaptive_topk=adaptive,
    denoiser=denoiser,
    reranker=reranker,
)

# 执行检索
results, latency = await pipeline.retrieve(
    query="什么是企业级RAG系统？",
    knowledge_base_id=1,
    top_k=5,
    parent_retrieval_mode="dynamic",
    dynamic_expand_n=2,
)
```

## 已知限制

1. **Reranker 模型依赖**：需要 sentence-transformers，可能因模型加载失败回退到词重叠评分
2. **jieba 分词库**：需预先初始化，首次调用可能有延迟
3. **PostgreSQL 连接**：BM25 检索需要数据库连接，需确保连接池配置正确

## 延迟基准（目标）

| 组件 | 目标延迟（P95） |
|------|----------------|

| Vector | < 300ms |

## 注意事项

1. **向量检索器初始化**：需要 `BgeM3EmbeddingService` 和 `ChromaVectorStore`
2. **BM25 配置**：需确保 `content_tsv` 列已正确索引，使用 `pg_jieba` 配置
3. **父文档模式**：
   - `off`：不扩展，适合小型知识库
   - `dynamic`：动态扩展相邻 chunks，推荐
   - `physical`：需要双层 chunks（Parent + Child）
4. **自适应深度**：
   - `min_k`：确保返回最少结果数
   - `max_k`：限制最大返回数
   - `cliff_factor`：控制断崖检测敏感度
