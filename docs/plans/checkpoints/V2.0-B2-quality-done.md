## 已完成功能

### NLI 幻觉检测
- 交叉编码器：cross-encoder/nli-deberta-v3-base
- 中英文混合句子分割
- 三分类：entailment / neutral / contradiction
- faithfulness_score 计算

### 置信度评估
- 综合 NLI 忠实度 + 检索分数
- 置信度等级：high（>0.8）、medium（0.5-0.8）、low（<0.5）

### 引用验证
- 正则提取 \[ID:x] 标记
- NLI 验证每句话的 entailment
- 移除虚假引用

### 验证 Pipeline
- PASS/FILTER/RETRY/REFUSE 决策
- 决策规则：confidence ≥ 0.5, faithfulness ≥ 0.3, citation ≥ 0.5
