## 1. 系统概述

本系统是基于 RAG（检索增强生成）技术的企业知识库问答平台，支持多模态文档解析、混合检索、智能问答和引用溯源。

### 核心功能

- **智能问答**：基于知识库的 RAG 问答，支持流式输出、多轮对话
- **引用溯源**：回答自动标注来源文档和位置（`[ID:x]`）
- **文档处理**：支持 TXT、MD、PDF、DOC、DOCX、XLS、XLSX、PPT、PPTX、PNG、JPG、JPEG
- **知识库管理**：创建、编辑、删除知识库，支持 RBAC 权限控制
- **文件夹同步**：自动同步本地文件夹到知识库
- **检索质量看板**：可视化检索日志和用户反馈

### 技术栈

- 前端：React + Vite + TypeScript + Tailwind CSS
- 后端：FastAPI + SQLAlchemy + PostgreSQL + ChromaDB
- AI：混合检索（BM25 + 向量）、Reranker、LLM 生成

## 4. 知识库管理

### 创建知识库

1. 进入"知识库"页面
2. 点击"创建知识库"按钮
3. 填写知识库信息：
   - **知识库名称**：例如"产品文档"、"技术手册"
   - **描述**：可选，简要说明知识库内容
   - **分块策略**：选择合适的分块方式
     - `chinese_recursive`：中文递归分块（推荐）
     - `recursive`：通用递归分块
     - `fixed`：固定大小分块
   - **分块大小**：默认 800，可根据文档调整
   - **分块重叠**：默认 100，建议为分块大小的 10-15%
4. 点击"创建"

### 编辑知识库

1. 在知识库列表中，点击目标知识库的"编辑"按钮
2. 修改名称、描述或分块策略
3. 点击"保存"

### 删除知识库

> ⚠️ 删除知识库将同时删除所有相关文档和对话记录！

1. 点击知识库的"删除"按钮
2. 确认删除操作

### 知识库权限（RBAC）

系统支持 4 种角色权限：

| 角色 | 权限说明 |
|------|----------|
| admin | 完全控制：创建、编辑、删除知识库，上传文档 |
| editor | 编辑权限：上传文档、编辑 chunks、删除文档 |
| viewer | 只读权限：搜索和问答，不能修改内容 |
| contributor | 贡献权限：上传文档，不能删除 |

在用户管理中为用户分配知识库权限。

## 5. 文档上传

### 支持的文档格式

| 类别 | 支持格式 | 单文件上限 |
|------|----------|------------|
| 文本 | TXT, MD | 200 MB |
| Word | DOC, DOCX | 200 MB |
| Excel | XLS, XLSX | 200 MB |
| PowerPoint | PPT, PPTX | 200 MB |
| PDF | PDF | 200 MB |
| 图片 | PNG, JPG, JPEG | 200 MB |

### 上传文档

1. 进入"文档上传"页面
2. 选择目标知识库
3. 点击"上传文档"或拖拽文件到上传区域
4. 等待文档上传完成
5. 文档将自动解析和向量化

### 文档解析状态

上传后，文档有以下状态：

| 状态 | 说明 |
|------|------|
| pending | 等待解析 |
| parsing | 正在解析中 |
| vectorizing | 正在向量化 |
| completed | 完成，可用于问答 |
| failed | 解析失败，请检查文档格式 |

### 重新解析文档

如果文档解析失败或需要更新：

1. 在文档列表中，找到目标文档
2. 点击"重新解析"按钮
3. 系统将重新解析并向量化文档

### 批量上传

1. 在"文档上传"页面
2. 选择多个文件
3. 点击"批量上传"
4. 系统将按顺序处理所有文件

## 6. RAG 智能问答

### 问答界面

1. 进入"问答对话"页面
2. 在左侧选择目标知识库
3. 在输入框中输入问题
4. 点击"发送"或按 Enter 键

### 对话历史

- 系统自动保存对话历史
- 点击左侧对话列表查看历史对话
- 可以继续历史对话

### 引用溯源

回答中的 `[ID:x]` 表示引用来源：

- **ID**：引用的 chunk 编号
- **来源**：点击引用可查看原始文档内容
- **位置**：显示文档名称和 chunk 索引

### 检索策略选择

系统支持多种检索策略：

| 策略 | 说明 | 适用场景 |
|------|------|----------|
| smart | 智能策略：自动选择最佳方案 | 通用场景 |
| precise | 精准策略：高质量、慢 | 准确性要求高 |
| fast | 快速策略：低延迟、低质量 | 快速响应 |
| deep | 深度策略：大量检索 | 复杂查询 |

### 查询扩展

可选择以下查询扩展模式：

- **规则扩展**：基于关键词规则扩展
- **LLM 扩展**：使用 LLM 生成相关问题
- **混合扩展**：结合规则和 LLM

## 8. 文件夹同步

### 配置同步

1. 进入"文件夹同步"页面
2. 点击"添加同步配置"
3. 填写配置：
   - **知识库**：选择目标知识库
   - **本地路径**：输入要同步的本地文件夹路径
   - **同步模式**：
     - `monitor`：监控模式，文件变化时自动同步
     - `manual`：手动模式，需要手动触发
   - **同步间隔**：轮询间隔（分钟），默认 30 分钟
4. 点击"保存"

### 手动触发同步

1. 在同步列表中，找到目标配置
2. 点击"立即同步"按钮
3. 系统将扫描文件夹并同步所有新文件

### 同步日志

- 系统记录每次同步的详细日志
- 可查看同步状态：成功、失败、跳过的文件

## 10. 常见问题

### Q: 文档上传后无法问答？

A: 请检查以下事项：
1. 文档解析状态是否为"completed"
2. 知识库是否已选择
3. 文档内容是否与问题相关

### Q: 如何提升检索质量？

A: 建议优化以下方面：
1. 使用合适的分块大小和重叠
2. 优化文档标题和结构
3. 使用 Reranker 重排序
4. 调整检索策略和 TopK 参数

### Q: 支持哪些文档格式？

A: 支持以下格式：
- 文本：TXT, MD
- Office：DOC, DOCX, XLS, XLSX, PPT, PPTX
- PDF：PDF（支持 OCR 扫描文档）
- 图片：PNG, JPG, JPEG
