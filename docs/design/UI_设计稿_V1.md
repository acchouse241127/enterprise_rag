## 四、交互规范

### 4.1 加载状态

```
- 按钮加载: 按钮内显示旋转图标，禁用点击
- 页面加载: 显示骨架屏 (Skeleton)
- 列表加载: 显示加载中占位符
```

### 4.2 错误处理

```
- 表单错误: 输入框下方显示红色错误文字
- API错误: 页面顶部显示 Toast 提示
- 网络错误: 显示重试按钮
```

### 4.3 确认对话框

```
删除操作: "确定删除吗？此操作不可恢复。"
危险操作: 红色确认按钮
```
