## 一、系统概述

### 1.1 系统简介

Enterprise RAG 是一个企业级知识库问答系统，基于 RAG（Retrieval-Augmented Generation）技术实现智能问答。系统支持：

- 多知识库管理
- 多格式文档上传与解析（PDF、Word、Excel、PPT、TXT、MD、图片等）
- 网页内容导入
- 流式问答与引用展示
- 对话导出与分享
- 检索质量看板与反馈
- 文件夹同步
- 双因素认证（TOTP）

### 1.2 技术架构

```
┌─────────────────────────────────────────────────────────┐
│                    前端（Streamlit）                      │
│  登录 | 问答 | 知识库 | 文档 | 对话 | 看板 | 任务          │
└─────────────────────────────────────────────────────────┘
                           │ REST API
                           ▼
┌─────────────────────────────────────────────────────────┐
│                    后端（FastAPI）                        │
│  认证 | 知识库 | 文档 | 问答 | 检索 | 对话 | 任务          │
└─────────────────────────────────────────────────────────┘
         │              │              │
         ▼              ▼              ▼
    PostgreSQL     ChromaDB        LLM API
    （关系数据）    （向量存储）    （智能生成）
```

### 1.3 系统要求

- Python 3.11+
- PostgreSQL 14+
- Redis（可选，用于异步任务队列）
- 8GB+ 内存（推荐）

## 二、快速开始

### 2.1 启动后端

```powershell
cd backend
pip install -r requirements.txt
# 配置 .env 文件
python main.py
```

后端默认运行在 `http://localhost:8000`

### 2.2 启动前端

```powershell
cd frontend
pip install -r requirements.txt
streamlit run 系统介绍.py
```

前端默认运行在 `http://localhost:8501`

### 2.3 默认账号

| 用户名 | 密码 | TOTP | 说明 |
|--------|------|------|------|
| admin | password123 | 无需 | 管理员账号 |
| admin_totp | password123 | 需要 | TOTP 测试账号 |

## 三、前端功能详解

### 3.1 登录页面（1_登录）

**功能说明：** 用户认证入口，支持普通登录和 TOTP 双因素认证。

**操作步骤：**
1. 输入用户名和密码
2. 如已绑定 TOTP，输入 6 位验证码
3. 点击「登录」

**已登录状态：**
- 显示当前用户信息和角色
- 可点击「绑定 TOTP」开启双因素认证
- 可点击「退出登录」安全登出

### 3.2 问答对话（2_问答对话）

**功能说明：** 核心功能页面，进行 RAG 智能问答。

**操作步骤：**
1. 选择知识库
2. 输入问题
3. 查看流式回答和引用来源
4. 可对回答进行反馈（有用/无用）

**注意事项：**
- 问题应尽量具体
- 引用来源可点击展开查看原文

### 3.3 知识库管理（3_知识库管理）

**功能说明：** 创建、编辑、删除知识库，管理文档。

**操作步骤：**
1. 新建知识库：输入名称和描述，点击「创建」
2. 编辑知识库：点击「编辑」修改名称/描述
3. 管理文档：点击「管理」查看文档列表
4. 删除知识库：点击「删除」（会删除所有文档）

**文档操作：**
- 重新解析：重新处理文档内容
- 预览：查看解析后的文本
- 下载：下载原始文件
- 删除：删除文档

### 3.4 文档上传（4_文档上传）

**功能说明：** 上传文件或从网址导入文档。

**文件上传：**
1. 选择知识库
2. 拖拽或选择文件（支持多选）
3. 点击「开始上传」

**URL 导入：**
1. 在「从网址导入」区域输入 URL
2. 点击「导入」
3. 系统自动抓取网页正文并创建文档

**支持的文件格式：**
- 文本：TXT, MD
- 文档：PDF, DOCX, XLSX, PPTX
- 图片：PNG, JPG

**版本管理：**
- 上传同名文件会创建新版本
- 可切换激活不同版本

### 3.5 对话管理（5_对话管理）

**功能说明：** 管理历史对话，支持导出和分享。

**操作步骤：**
1. 按知识库筛选对话
2. 导出为 MD/PDF/DOCX
3. 生成分享链接（7 天有效期）
4. 预览对话消息
5. 删除不需要的对话

**分享功能：**
- 点击「分享」生成链接
- 他人可通过链接免登录查看对话内容
- 可随时取消分享

### 3.6 文件夹同步（6_文件夹同步）

**功能说明：** 自动同步本地文件夹到知识库。

**配置步骤：**
1. 选择知识库
2. 输入文件夹路径
3. 设置文件过滤规则（通配符）
4. 保存配置

**同步方式：**
- 手动触发
- 定时同步

### 3.7 检索质量看板（7_检索质量看板）

**功能说明：** 查看检索统计、日志和反馈。

**核心指标：**
- 总查询数
- 平均最高分
- 平均响应时间
- 无用反馈率
- 问题样本数

**日志操作：**
- 筛选：按知识库、反馈状态
- 反馈：标记有用/无用
- 标记问题样本：🔖 按钮
- 查看详情：点击展开

**统计视图：**
- 每日查询趋势图
- 按知识库统计表
- 问题样本列表

### 3.8 异步任务（8_异步任务）

**功能说明：** 查看后台任务状态。

**任务类型：**
- 文档解析
- 文档向量化
- 文件夹同步
- 对话导出

**操作：**
- 查看任务详情
- 取消进行中的任务
- 按实体查询关联任务

### 3.9 知识库编辑（9_知识库编辑）

**功能说明：** 高级编辑功能，修改文档内容和分块参数。

**功能：**
- 编辑文档内容
- 调整分块参数
- 重新分块
- 批量操作

### 3.10 分享查看（10_分享查看）

**功能说明：** 免登录查看分享的对话内容。

**访问方式：**
- 直接打开分享链接（带 token 参数）
- 或在前端访问 /分享查看 页面并输入 token

## 四、后端 API 参考

### 4.1 认证接口

#### POST /api/auth/login
登录获取 JWT token。

**请求体：**
```json
{
  "username": "admin",
  "password": "password123",
  "totp_code": "123456"  // 可选
}
```

**响应：**
```json
{
  "code": 0,
  "message": "success",
  "data": {
    "access_token": "eyJ...",
    "token_type": "bearer"
  }
}
```

#### GET /api/auth/me
获取当前用户信息（需认证）。

#### POST /api/auth/logout
服务端登出（需认证）。

#### POST /api/auth/totp/setup
生成 TOTP 密钥。

#### POST /api/auth/totp/verify
校验并绑定 TOTP。

### 4.2 知识库接口

#### GET /api/knowledge-bases
获取知识库列表。

#### POST /api/knowledge-bases
创建知识库。

#### GET /api/knowledge-bases/{id}
获取知识库详情。

#### PUT /api/knowledge-bases/{id}
更新知识库。

#### DELETE /api/knowledge-bases/{id}
删除知识库。

### 4.3 文档接口

#### POST /api/knowledge-bases/{id}/documents
上传单个文件。

#### POST /api/knowledge-bases/{id}/documents/batch
批量上传文件。

#### POST /api/knowledge-bases/{id}/documents/from-url
从 URL 导入文档。

#### GET /api/documents/{id}
获取文档详情。

#### DELETE /api/documents/{id}
删除文档。

#### POST /api/documents/{id}/reparse
重新解析文档。

### 4.4 问答接口

#### POST /api/qa/ask
非流式问答。

#### POST /api/qa/stream
流式问答（SSE）。

### 4.5 检索接口

#### GET /api/retrieval/logs
获取检索日志列表。

#### GET /api/retrieval/logs/{id}
获取单条日志详情。

#### POST /api/retrieval/feedback
添加反馈。

#### POST /api/retrieval/feedback/{id}/mark-sample
标记/取消问题样本。

#### GET /api/retrieval/stats
获取检索统计。

#### GET /api/retrieval/stats/by-knowledge-base
按知识库统计。

#### GET /api/retrieval/samples
获取问题样本列表。

### 4.6 对话接口

#### GET /api/conversations
获取对话列表。

#### GET /api/conversations/{id}/messages
获取对话消息。

#### POST /api/conversations/{id}/share
开启分享。

#### DELETE /api/conversations/{id}/share
关闭分享。

#### GET /api/conversations/share/{token}
免登录查看分享（无需认证）。

### 4.7 任务接口

#### GET /api/tasks
获取任务列表。

#### GET /api/tasks/{id}
获取任务详情。

#### POST /api/tasks/{id}/cancel
取消任务。

#### GET /api/tasks/entity/{type}/{id}
按实体查询任务。

## 五、常见问题

### Q1: 登录后显示「登录已过期」？
**A:** Token 已过期，请重新登录。默认 token 有效期为 24 小时。

### Q2: 文档上传后一直显示「处理中」？
**A:** 检查后端日志，确认 Worker 是否正常运行。可能需要手动触发解析。

### Q3: 问答返回「无相关内容」？
**A:** 
- 确认知识库中有已向量化的文档
- 检查文档状态是否为「vectorized」
- 尝试使用更具体的问题

### Q4: 分享链接打开后显示「无效或已过期」？
**A:** 
- 分享链接可能已被取消
- 分享可能已超过有效期（默认 7 天）

### Q5: TOTP 绑定后无法登录？
**A:** 
- 确认时间同步正确
- 使用 admin_totp 账号测试
- 如无法解决，联系管理员重置 TOTP

### Q6: 如何查看 API 文档？
**A:** 启动后端后访问 `http://localhost:8000/docs`
