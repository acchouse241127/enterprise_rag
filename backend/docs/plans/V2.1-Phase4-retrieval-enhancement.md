### 4.1 问题分析器 (Query Analyzer)

```python
# backend/app/rag/query_analyzer.py
class QueryAnalyzer:
    """分析查询意图，识别是否需要特定模态"""

    CHART_KEYWORDS = ["图表", "趋势", "曲线", "柱状", "饼图", "折线"]
    TABLE_KEYWORDS = ["表格", "数据", "列", "行", "对比"]
    IMAGE_KEYWORDS = ["图片", "图像", "图示", "示意图", "截图"]

    def analyze(self, query: str) -> dict:
        return {
            "needs_chart": any(kw in query for kw in self.CHART_KEYWORDS),
            "needs_table": any(kw in query for kw in self.TABLE_KEYWORDS),
            "needs_image": any(kw in query for kw in self.IMAGE_KEYWORDS),
        }
```
